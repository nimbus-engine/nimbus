Code Example

Here is a simple "Hello World" application structure.

App.xml 
<App Name="Hello">
    <UI>
        <StackPanel VerticalAlignment="Center">
            <TextBlock Text="Hello World!" FontSize="24"/>
            <Button Content="Click Me" onClick="ShowAlert"/>
        </StackPanel>
    </UI>
    <Logic>
        <Handler Name="ShowAlert">
            <Alert Message="Welcome to Nimbus v3.0!"/>
        </Handler>
    </Logic>
</App>
